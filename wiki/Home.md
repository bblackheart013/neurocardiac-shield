## About

NeuroCardiac Shield is a multi-modal physiological monitoring platform that integrates brain (EEG) and heart (ECG) signal acquisition with machine learning risk prediction. Developed at New York University as an Advanced Project in Medical Device Software Development.

## Key Features

- 8-channel EEG monitoring (10-20 system)
- 3-lead ECG with PQRST analysis
- Real-time signal processing
- XGBoost + LSTM ensemble risk prediction
- Interactive web dashboard
